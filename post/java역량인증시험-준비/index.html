Java역량인증시험 준비

java 코딩연습

주력으로 사용하는 언어가 아니다보니 뭔가를 코딩하려고 할 때 매번 구글링 해야해서 속도가 굉장히 늦다.
실전에서 자주 등장하는 (등장 할 것으로 예상되는) 것들을 적어놓고 reference없이 코딩 할 수 있도록 연습하자.
새로운 문제가 생각나면 계속 업데이트 하자

String 관련

비교, 검색, 치환, index, 추가, 삭제, loop, shift 하는 방법 익히기

  • String 오른쪽, 왼쪽 shift 하기, queue방식 shift 또는 새로운 자리에는 '0'이 채워지도록 shift

    String tmpStr = "abcdef123456";
// left shift
String leftShiftStr = tmpStr.subString(1) + tmpStr.chatAt(0);
System.out.println("[left shift] : " + leftShiftStr);

  • String 길이 구하고, String -> ArrayList로 변환하기

  • String 과 String 합치기. 각 String 에서 한글자씩 뽑아서 합치기, 한 String 의 홀수 + 다른 String 의 짝수 합치기

  • String을 특정 문자열 기준으로 자르기, 자른 문자열의 N번째 문자열 취하기. 2개이상의 특정문자열로 자르기

  • String을 for loop에서 한글자씩 취해서 print하고 다시 원래 String으로 합치기

  • String잘라서 collection객체만들기

  • String 한글자씩 잘라서 정렬하기(ace, desc), 숫자끼리 문자끼리 정렬하기

  • String의 제일 앞뒤 빈칸 제거하기, 중간에 있는 빈칸만 제거하기

  • String의 각각의 글자를 ascii값으로 변환하고, ascii에서 다시 String으로 변환해서 합치기

  • String <-> String[] 으로 변환하기

  • String <-> Integer 로 변환
