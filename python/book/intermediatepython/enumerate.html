Enumerate¶

Enumerate 는 python의 built-in 함수입니다. 이것이 얼마나 유용한지는 한 줄로 표현할 수 없습니다. 대부분의 초심자들, 그리고 심지어 몇몇 숙련된 개발자들도 아직 Enumerate 에 대해 잘 알지 못합니다. Enumerate는 여러분이 loop 를 수행하도록 도와주고, 자동으로 횟수를 증가시켜주는 counter를 가지고 있습니다. 아래 예제를 살펴보겠습니다.

for counter, value in enumerate(some_list):
    print(counter, value)

이것이 enumerate 의 전부가 아닙니다. enumerate 는 추가의 option과 함께 사용가능하고, 이러한 옵션이 enumerate 를 훤씬 유용하게 만들어줍니다..

my_list = ['apple', 'banana', 'grapes', 'pear']
for c, value in enumerate(my_list, 1):
    print(c, value)

# Output:
# 1 apple
# 2 banana
# 3 grapes
# 4 pear

우리는 추가 인자를 통해 enumerate 에게 어떤 index부터 시작할것인지 알려줄 수 있습니다. 또한 여러분은 list를 이용하여 index와 item으로 구성되는 tuple을 만들수 있습니다. 아래는 예제입니다.

my_list = ['apple', 'banana', 'grapes', 'pear']
counter_list = list(enumerate(my_list, 1))
print(counter_list)
# Output: [(1, 'apple'), (2, 'banana'), (3, 'grapes'), (4, 'pear')]
